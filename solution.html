\( \int_{-\infty}^{+\infty} x \frac{\alpha}{2} * e^{- \alpha * | x - \beta |} = \frac{\alpha}{2} int_{-\infty}^{\beta} x * e^{ \alpha * ( x - \beta) } + \int_{\beta}^{+\infty} x * e^{- \alpha * (x - \beta) }) \)

Посчитаем первый:

\( t=x-\beta => x=t+\beta \)
