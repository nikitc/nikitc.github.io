\( \int_{-\infty}^{+\infty} x \frac{\alpha}{2} * e^{- \alpha * | x - \beta |} = \alpha/2 \int_{-\infty}^{+\infty} x * e^{- \alpha * | x - \beta |} \)

Применим интегрирование по частям:

\( u=x => du = x' dx = dx\)

\( dv=e^{- \alpha * | x - \beta |} => v = int e^{- \alpha * | x - \beta |} = \frac{1}{-\alpha} * e^{- \alpha * | x - \beta |}\)

получим

\( \frac{x}{-\alpha} * e^{- \alpha * | x - \beta |} + \frac{1}{\alpha} \int_{-\infty}^{+\infty} e^{- \alpha * | x - \beta |} dx\)

Посчитаем второй интеграл

\( \int_{-\infty}^{+\infty} e^{- \alpha * | x - \beta |} dx = \int_{-\infty}^{+\beta} e^{\alpha * ( x - \beta )} dx + \int_{\beta}^{+\infty} e^{- \alpha * ( x - \beta )} dx = 1 - 0 + 0 - 1 = 0 \)
